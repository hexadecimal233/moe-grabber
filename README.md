# 萌备自动化抢号工具

本工具会在指定时间执行萌备抢号任务。

支持的邮箱服务为 Proton Mail，也可自己使用 IMAP 协议连接其他邮箱服务。

## 特性

- [x] 全自动化数据提交

- [x] 支持多次重试，遇到错误自动重试

- [x] 验证码识别和邮箱验证码获取

- [ ] 更高精度的 OCR 模型

- [ ] 遇到当前萌备已经被抢则会试着抢下一个号

- [ ] 支持更多邮箱服务

## 运行

1. 安装依赖

```bash
pdm install
```

2. 配置 `config.json` 文件，填写相关信息。

3. 运行脚本

```bash
pdm run main.py
```
